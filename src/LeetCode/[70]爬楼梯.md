# 70. 爬楼梯

## 题目

假设你正在爬楼梯。需要 *n* 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**注意：**给定 *n* 是一个正整数。



## 思路

### 遍历

n对应的值为(n - 1)对应的值加上(n - 2)对应的值。遍历可求解。

#### 时间复杂度

O(n)

#### 空间复杂度

O(1)

#### 代码

```javascript
/**
 * @param {number} n
 * @return {number}
 */
var climbStairs = function(n) {
    let val1 = 1;
    let val2 = 1;
    for (let i = 2; i <= n; i++) {
        let current = val2 + val1;
        val1 = val2;
        val2 = current;
    }

    return val2;
};
```
