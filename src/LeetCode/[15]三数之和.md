# 15. 三数之和

## 题目

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a，b，c ，使得 a + b + c = 0 ？请你找出所有满足条件且不重复的三元组。

注意：答案中不可以包含重复的三元组。

## 思路

### 排序双指针



#### 时间复杂度



#### 空间复杂度



#### 代码

```javascript
/**
 * @param {number[]} nums
 * @return {number[][]}
 */
var threeSum = function(nums) {
    const len = nums.length;

    nums.sort((a, b) => a - b);
    if (nums[0] > 0) {
        return [];
    } else if (nums[len - 1] < 0) {
        return [];
    }

    const list = [];
    
    for (let i = 0; i < len - 2; i++) {
        const f = nums[i];
        if (f > 0 || f === nums[i - 1]) {
            continue;
        }

        let start = i + 1;
        let end = len - 1;
        while (start < end) {
            const s = nums[start];
            if (start !== i + 1 && s === nums[start - 1]) {
                start++;
                continue;
            }

            const t = nums[end];
            if (t === nums[end + 1]) {
                end--;
                continue;
            }

            const r = -(s + t);
            if (r === f) {
                list.push([f, s, t]);
                start++;
            } else if (r > f) {
                start++;
            } else {
                end--;
            }
        }
    }
    
    return list;
};
```



### 转两数之和

#### 空间复杂度



#### 代码

```javascript
/**
 * @param {number[]} nums
 * @return {number[][]}
 */
var threeSum = function(nums) {
    const len = nums.length;
    
    const keys = new Set();
    const list = [];
    let hasZero = false;
    
    for (let i = 0; i < len - 2; i++) {
        const f = nums[i];

        const sum = -f;
        const tempSet = new Set();
        let index = i + 1;
        while(index < len) {
            const val = nums[index];
            if (f === 0 && val === 0 && hasZero) {
                index++;
                continue;
            }

            const s = sum - val;
            if (tempSet.has(val)) {
                let key = Math.min(f, val, s) + ',' + Math.max(f, val, s);
                if (!keys.has(key)) {
                    keys.add(key);
                    list.push([f, val, s]);
                    if (f === 0 && val === 0) {
                        hasZero = true;
                    }
                }
            } else {
                tempSet.add(s);
            }

            index++;
        }
    }
    
    return list;
};
```
