# 19. 删除链表的倒数第N个节点

## 题目

给定一个链表，删除链表的倒数第 *n* 个节点，并且返回链表的头结点。

**说明：**

给定的 *n* 保证是有效的。

**进阶：**

你能尝试使用一趟扫描实现吗？

> **测试**
>
> 给定一个链表: 1->2->3->4->5, 和 n = 2.
>
> 当删除了倒数第二个节点后，链表变为 1->2->3->5.

## 思路

### 转数组法

#### 代码

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @param {number} n
 * @return {ListNode}
 */
var removeNthFromEnd = function(head, n) {
    let list = [];

    let c_node = head;

    while (c_node) {
        list.push(c_node);
        c_node = c_node.next;
    }

    let prev_node = list[list.length - n - 1];
    let delete_node = list.splice(-n, 1)[0];
    if (prev_node) {
        prev_node.next = delete_node.next;
        return head;
    } else {
        return delete_node.next;
    }
};
```

```javascript
/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @param {number} n
 * @return {ListNode}
 */
var removeNthFromEnd = function(head, n) {
    let r_code = head;
    let c_node = head;
    
    let s = 0;

    while (c_node.next) {
        if (s === n) {
            r_code = r_code.next;
        } else {
            s++;
        }

        c_node = c_node.next;
    }

    if (s === n) {
        r_code.next = r_code.next.next;
        return head;
    } else {
        return r_code.next;
    }
};
```
